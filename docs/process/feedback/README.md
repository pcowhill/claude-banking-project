# Human feedback (verbatim)

Each session saves the human's pasted review feedback here **verbatim, before
acting on it**, as:

```
FEEDBACK_vX.Y_YYYY-MM-DD_HHMM.md
```

The raw/verbatim block in each file must **never** be edited or paraphrased after
it is saved. See `docs/process/HUMAN_FEEDBACK_LOG.md` for the required template
and the index of all feedback files.
