# Customer app images (drop-in)

Marketing photos are **not committed**. The UI renders a branded gradient
placeholder until a real file exists here, then uses the file automatically —
no code change required.

Expected file names (see `assets/prompts/IMAGE_GENERATION_PROMPTS.md`):

- `hero-family.jpg` — homepage hero
- `feature-professional.jpg`
- `feature-small-business.jpg`
- `product-card.jpg`
- `security-protection.jpg`

Use original or properly licensed images only. Never use real bank logos,
copyrighted assets, or real-person likenesses.
